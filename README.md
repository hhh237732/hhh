# hhh 微内核知识库

一个基于**微内核架构**设计的知识库，用于存储代码片段、技术文献、研究笔记等各类知识。

## 架构概览

本知识库采用微内核（Microkernel）架构：

- **核心（Core）**：提供最小化的基础能力——知识条目的统一接口定义、注册表与元数据规范。
- **插件（Plugins）**：每种知识类型以独立插件形式扩展，互不耦合，可按需增减。
- **知识库（Knowledge）**：实际的知识条目，按插件分类存放。

```
hhh/
├── core/               # 微内核：注册表、接口规范、元数据 Schema
├── plugins/            # 插件层：各类知识类型的扩展模块
│   ├── code/           #   代码知识插件
│   └── literature/     #   文献知识插件
├── knowledge/          # 知识条目（数据层）
│   ├── code/           #   代码片段与示例
│   └── literature/     #   文献、论文、书籍笔记
└── docs/               # 架构说明与贡献指南
```

## 快速开始

1. 浏览 [架构文档](docs/architecture.md) 了解整体设计。
2. 查看 [核心注册表](core/registry.md) 了解现有知识条目。
3. 参考对应插件的 `README.md` 添加新知识条目。

## 扩展新插件

在 `plugins/` 下新建目录，并按 [插件规范](core/plugin-spec.md) 创建 `README.md` 与 `index.md` 即可接入新的知识类型。
